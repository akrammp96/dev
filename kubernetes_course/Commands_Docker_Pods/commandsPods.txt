____How to pass the arguments to docker container ??____

As said in the dockerfile commands doc, the CMD could be changed

with docker run now with the pod definition we can specify 

the args when running the container from a specified image. 


To modify the ENTRYPOINT we should use: 
command: ["<command>"]




How to edit pods: 


kubectl edit pod webapp

kubectl delete pod webapp

"after making changes"
kubectl create -f /tmp/kubectl-edit-ccvrq.yaml


Second way to do it
kubectl get pod webapp -o yaml > my-new-pod.yaml
vi my-new-pod.yaml
